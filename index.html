I don't want the stats

I don't want a second Pokemon

Remove one {pokemon} inside the "pokemon": []

"pokemon": [
  {
    "name": "Pikachu",
    "formName": "Normal",
  },
  {
    "name": "Pikachu",
    "formName": "Shiny",
  }
],

like this

"pokemon": [
  {
    "name": "Pikachu",
    "formName": "Normal",
  }
],

    "name": "Master Passive",

  }

"passivesMaster": [],
